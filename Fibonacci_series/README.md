# Generating Fibonacci Numbers Using a Local MATLAB Function
The program accepts a number n and returns a vector containing the first n Fibonacci numbers. 

- It sets n = 10 to specify how many Fibonacci numbers to generate.
- Calls the local function fibonacci(n) to compute the first n terms of the sequence.
- Displays a message indicating how many Fibonacci numbers will be shown.
- Prints the resulting Fibonacci vector in the command window.
